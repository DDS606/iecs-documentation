# RACI Matrix

The RACI matrix defines the roles and responsibilities for key activities in the IECS project.

R = Responsible (does the work)  
A = Accountable (owns the result)  
C = Consulted (provides input)  
I = Informed (kept updated)

| Activity | Project Manager | Backend Dev | Frontend Dev | IoT Engineer | QA Engineer |
|----------|----------------|-------------|--------------|--------------|-------------|
| Project Planning | A | I | I | I | I |
| System Architecture | C | R | C | C | I |
| Backend Development | I | R | C | C | I |
| Frontend Development | I | C | R | I | I |
| Sensor Integration | I | C | I | R | I |
| System Testing | I | C | C | C | R |
| Project Delivery | A | R | R | R | C |
