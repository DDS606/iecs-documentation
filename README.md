# Intelligent Energy Control System (IECS)

This repository contains the project documentation for the **Intelligent Energy Control System (IECS)** using the **Documentation as Code** approach.

The documentation is written in Markdown and generated as a website using **MkDocs**.

---

# Project Overview

The Intelligent Energy Control System (IECS) is a smart platform designed to monitor and optimize energy consumption for households and businesses.

The system combines:

- smart energy sensors
- data analytics
- a web-based dashboard

to provide users with real-time insights into their energy usage and help improve energy efficiency.
