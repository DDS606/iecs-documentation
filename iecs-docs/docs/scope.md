# Project Scope

## In Scope

The IECS project will include:

- Development of smart energy monitoring sensors
- A backend system for collecting and processing energy data
- A web dashboard for monitoring energy consumption
- Data analytics for identifying energy usage patterns
- Visualization tools for energy insights

## Out of Scope

The following items are not included in the project:

- Large-scale power infrastructure development
- Manufacturing of energy equipment at industrial scale
- Integration with every possible external energy provider

## Deliverables

- Functional smart sensor prototype
- Web-based energy monitoring dashboard
- Backend data processing system
- Documentation and system testing results
